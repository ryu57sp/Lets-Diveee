# Let's Diveee!!!
<img src="https://github.com/ryu57sp/LetsDiveee/blob/main/app/assets/images/top.jpg" width="100%">

## サイト概要
このサイトは海が好きな人が集まるダイビングに特化したサイトです。

会員登録済みのユーザーは投稿機能を使って、自身の情報を共有・発信できます。

また、他のSNS同様にフォロー・フォロワー機能・いいね機能・コメント機能・DM機能を用いて他の会員登録済みのユーザーと関わることができます。

ライセンスは持ってないけどダイビングに興味がある方や海の写真を見るのが好きな方も会員登録なしでサイトを閲覧することが可能です。

### サイトテーマ
ダイビング（海）に特化したSNSサイト

### テーマを選んだ理由
僕自身、高校卒業後にハワイに留学したことがきっかけで海が大好きになり、PADIインストラクターの資格を取りました。

同時に、初めてイルカやウミガメと泳いだ時の感動が忘れられず、こういった海の魅力（水中の世界）を多くの人に伝えたいという気持ちから水中写真を本格的に始めました。

現在はTwiiterをメインに海の魅力を発信していますが、発信者とダイバーという両方の観点から『もっとダイビング（海）の情報が詰まったサイトがあったらな』という思いがあり、このテーマを選びました。

現状、SNSに載っているダイビング関連の投稿は写真がメインのため、ダイビングの詳細情報までは含まれていないものがほとんどです。

なので、ポイントやショップ選びなどをする際には、SNSとは別でいくつかのサイトを見て情報を仕入れる必要があります。

そこで、SNSの基本機能はそのまま且つそれらの情報も一緒に共有できるプラットフォームを作れば、ダイビングの情報が詰まったサイトになるだけでなくコミュニティーの活性化にもつながると考えました。

また、ジャンルを絞ることでユーザーにとって情報量の多すぎない使いやすいサイトを実現できると考えました。

### ターゲットユーザー
ダイバー・ダイビングに興味がある人・海が好きな人

### 主な利用シーン
#### 🤿 ダイバー
- 他のダイバーさんと繋がりたい時
- シーズンや水温などの情報を得たい時
- どこのポイントに行こうか迷っている時
- 流行りのスポットや大物情報を知りたい時
- ポイント近くのショップ情報を知りたい時
- ダイビングで撮影した写真と情報を思い出として残したい時
- 多くの人に向けて海の魅力を発信したい＆海に行くきっかけを与えたい人
#### 🏝 ダイビングに興味がある人・海が好きな人
- ダイビングの世界を見てみたい時
- どのランクまで取ろうか迷っている時
- 綺麗な海の写真を見て癒やされたい時
